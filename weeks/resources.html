En esta sección encontraremos información y recursos importantes para el aprendizaje de los principales vendors en Cloud

Los recursos acá mostrados algunos son gratuitos y otros pagos. Y están destinados a un nivel fundacional.

Amazon Web Services

El primer nivel de cloud para AWS es llamado Cloud practitioner, dentro del cual conoceremos los principales servicios que ofrece AWS, para poder lograr esta certificación debemos estar en la capacidad de dado un problema, poder conocer los servicios que pueden aportar a solucionar dicha problematica.

Lista de recursos:

  • Contiene distintos tipos de suscripción para un aprendizaje autonomo
  • AWS Cloud Practitioner Essentials
  • Learning Path Cloud Foundations
  • Developer Learning Path
  • Networking Learning plan
  • Data analytics Learning plan
  • Cloud Practitioner AWS Freecodecamp

Google Cloud Platform

GCP es la nube ofrecida por Google y es la más reciente pero con mucho potencial por delante.

Lista de recursos

  • Pagina Principal Trainings
  • Catalogo de cursos gratuitos y pagos
  • Google Cloud Engineer Course
  • Google Cloud Platform
  • Badges GCP
  • Learning path infraestructure, primer curso gratis

Microsoft Azure

La última pero no menos importante, Azure es la nube pública ofrecida por Microsoft la cual tiene sus pros y sus contras , en referencia a sus competidores, siendo estas 3 las que lideran el mercado actualmente. AZ-900 es su primer certificado fundacional, que garantiza a los que logren obtener la certificación de conocer y dar respuesta a los principales problemas utilizando la plataforma de Azure.

Lista de recursos

  • Cloud Concepts
  • Lista de Reproducción examen AZ-9000
  • Azure Fundamentals Freecodecamp
  • Training Day 1 Azure 2022
